One registry, every server

Cards are global, not locked to a single server. Submit a VTuber once and they become searchable from anywhere PoiiBot is installed. Search by name, filter by content type, or discover someone random.

Levels and Prestige

Two-track progression system

PoiiBot tracks Global (all servers, never resets) and Server (per-server, resets on prestige) levels separately. Both update from messages. Server level-ups post notifications and award Hana Coins (25 x new level).
/level-check
Check your current level with a progress bar. Optionally evaluate a target level, check another user, or switch between global and server scope.

Toggle server level-up notifications on or off. Exp still accumulates silently when off.

Set a dedicated channel for level-up messages. Without this, level-ups post in the user's active channel.

Remove the dedicated level-up channel.
/prestige
Available at Server Level 60. Resets server level to 0 for a permanent prestige badge and Hana Coin reward (1,000 x prestige number). Global level is never affected. Requires typing PRESTIGE to confirm. Max prestige is 5.

Privacy and Data Transparency

PoiiBot stores only what it needs to function. No message content is ever saved anywhere. Messages are read in real-time to handle commands, then discarded immediately. There is no message log, no conversation history, and no way for a database breach to expose anything you said.

What gets stored

  • Discord user IDs and server IDs (numbers only)
  • Economy data: Hana Coins, level, exp, fight records
  • Profile choices: bio, title, badge selections
  • Server settings: modlog channel, welcome channel, alert configs
  • Moderation records: warning reasons and who issued them
  • VTuber registry info (publicly submitted by users)

What never gets stored

  • Message content of any kind
  • Passwords or auth tokens
  • Email addresses, phone numbers, or real names
  • Payment or financial information
  • IP addresses or location data
  • Anything beyond what Discord itself already knows

In the event of a breach, the worst case exposure is Discord user IDs and game stats. That is information that is already visible on Discord itself and carries no real-world risk. There is no sensitive personal data in the database to expose.

If you want your data removed, reach out via /contact or @Hanako_Amaya_ on Twitter and it will be handled promptly.
